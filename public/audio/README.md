# Audio assets

Drop the intro ambient track here as **`intro.mp3`**.

## What to source

Search Pixabay / Freesound for:

> singing bowl wind chimes meditation

What works:

- Tibetan singing bowl drone, low and sustained
- Wind chimes layered softly on top
- 3–5 minutes long (loops automatically on the site)
- Gentle, no harsh peaks, no melody

The site fades the audio out as the cinematic intro ends. If this file is
missing the site silently continues — no error.
